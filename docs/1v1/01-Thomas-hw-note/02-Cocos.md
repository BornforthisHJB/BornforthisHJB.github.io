## 0. 讲解作业

1. 考点：变量

### 输出特定格式

```python
a = 1
b = 2
c = 3
```

问题1答案：

```python
print(a, b, c)
```

问题2答案：

```python
print(a)
print(b)
print(c)
```

### 交换果汁

两个杯子实现：

![image-20220821093010404](./02-Cocos.assets/image-20220821093010404.png)

那能否一个杯子实现呢？

```python
a = "果汁"
b = "可乐"
empty_cup = a
a = b
b = empty_cup
print("a:", a)
print("b:", b)
```
