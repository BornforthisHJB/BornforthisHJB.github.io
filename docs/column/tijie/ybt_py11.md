## 【题目描述】

某小组决定网购单价为1元至10元的十种笔记本各一本，求一共要花多少钱?

### 【输入】

(无)

### 【输出】

一行一个整数，表示答案。

## 【输入样例】

```python
(无)
```

## 【输出样例】

```python
(无)
```

## 解析

程序需要有一个循环变量从 1 变化到 10，将该变量设为 i，i 每次加到累加和 sum 中；变量 i 每增加 1 时，就和变量 sum 进行一次加法运算，变量 sum 记录的是累加的结果。

## 代码

```python
sum_num = 0
# 枚举 i 是 1～10 之间的数
for i in range(1, 11):
	sum_num += i
print(sum_num)
```
