## 【题目描述】

当飞船的速度大等于 7.91 千米/秒时飞船能成功飞行，给出飞船速度，请判断飞船是否能够成功飞行。

### 【输入】

一个实数 x，表示飞船速度是 x 千米/秒。

### 【输出】

如果飞船能成功飞行，则输出“飞船成功飞行！”，否则输出“飞船不能升空！”

## 【输入样例】

```python
8.01
```

## 【输出样例】

```python
飞船成功飞行！
```

## 代码

```python
v = float(input())  # 输入飞船速度
if v >= 7.91:  # 判断飞船能否飞行成功的速度
	print("飞船成功飞行!")
else:
	print("飞行不能升空!")
```
