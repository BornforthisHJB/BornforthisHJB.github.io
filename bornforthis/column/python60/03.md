## 列表

列表（list）作为 Python 中最常用的数据类型之一，是一个可增加、删除元素的可变（mutable）容器。

### 基本操作

创建 list 的方法非常简单，只使用一对中括号 `[]`。如下创建三个 list：

```python
empty = []
lst = [1, 'xiaoming', 29.5, '17312662388']
lst2 = ['001', '2019-11-11', ['三文鱼', '电烤箱']]
```
