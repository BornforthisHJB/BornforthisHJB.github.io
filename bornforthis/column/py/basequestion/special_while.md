### 1.2 Question：输出前十个可以整除 3 的数据

代码：

```python
divided = 100
divisor = 3
index = 0
while divided <= 200 and index < 10:
    print("index: {}".format(index))
    if divided % 3 == 0:
        print(f"{divided} 可以整除 3 。商为 {int(divided / 3)}")
    index += 1
```
