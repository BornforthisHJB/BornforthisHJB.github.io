# 解决方法

去掉项目名称中的所有空格。



# Why？

命令行或者代码中，包含空格时，其实大部分是语法或者命令：

```python
python file.py  # 这里的空格是下一个命令的编写
```

所以，文件中出现空格，代码实现却是和路径有关，所以程序将会无法分清。
