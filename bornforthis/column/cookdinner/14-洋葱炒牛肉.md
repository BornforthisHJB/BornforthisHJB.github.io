## 健康功效

- 牛肉:补脾胃、强筋骨、益气血
- 洋葱: 解毒
- 姜: 降逆止呕、化痰止咳、散寒解表

## 食材用料

| 牛肉 300克  | 洋葱 200克 |
| ----------- | ---------- |
| 姜 适量     | 生抽 适量  |
| 蚝油 适量   | 盐 适量    |
| 胡椒粉 适量 | 生粉 适量  |
| 青椒 一个   | 料酒 少许  |
