## 描述

给定一个字符，用它构造一个底边长5个字符，高3个字符的等腰字符三角形。

## 输入

输入只有一行， 包含一个字符。

## 输出

该字符构成的等腰三角形，底边长5个字符，高3个字符。

## 样例输入

```cpp
*
```

## 样例输出

```cpp
  *
 ***
*****
```

## 代码

```cpp
#include <iostream>
using namespace std;
int main()
{
    char c;
    cin >> c;
    cout << "  " << c << endl;
    cout << " " << c << c << c <<  endl;
    cout << c << c << c << c << c <<  endl;
    return 0;
}
```
