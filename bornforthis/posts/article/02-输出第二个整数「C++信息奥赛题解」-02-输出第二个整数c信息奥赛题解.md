## 描述

输入三个整数，把第二个输入的整数输出。

## 输入

只有一行，共三个整数，整数之间由一个空格分隔。整数是 32 位有符号整数。

## 输出

只有一行，一个整数，即输入的第二个整数。

## 样例输入

```cpp
123 456 789
```

## 样例输出

```cpp
456
```

## 代码

```cpp
#include <iostream>
using namespace std;
int main() {
    int a, b, c;
    cin >> a >> b >> c;
    cout << b << endl;
    return 0;
}
```
