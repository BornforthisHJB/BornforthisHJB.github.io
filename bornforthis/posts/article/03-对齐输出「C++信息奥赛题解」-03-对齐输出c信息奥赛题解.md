## 描述

读入三个整数，按每个整数占 8 个字符的宽度，右对齐输出它们。

## 输入

只有一行，包含三个整数，整数之间以一个空格分开。

## 输出

只有一行，按照格式要求依次输出三个整数，之间以一个空格分开。

## 样例输入

```cpp
123456789 0 -1
```

## 样例输出

```cpp
123456789        0       -1
```

## 代码

```cpp
#include <iostream>
#include <iomanip>
using namespace std;
int main()
{
    int a, b, c;
    cin >> a >> b >> c;
    cout << setiosflags(ios::right) << setw(8) << a << " "
    << setiosflags(ios::right) << setw(8) << b << " "
    << setiosflags(ios::right) << setw(8) << c << endl;
    return 0;
}
```
