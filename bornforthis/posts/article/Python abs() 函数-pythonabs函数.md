## 描述

abs() 函数返回数字的绝对值。

* * *

## 语法

以下是 `abs()` 方法的语法：

```
abs( x )
```

* * *

## 参数

*   x -- 数值表达式。

* * *

## 返回值

函数返回 x（数字）的绝对值。

* * *

## 实例

以下展示了使用 `abs()` 方法的实例：

```python
#!/usr/bin/python
print("abs(-45) : ", abs(-45))
print("abs(100.12) : ", abs(100.12))
print("abs(119L) : ", abs(119L))
```

以上实例运行后输出结果为：

```python
abs(-45) :  45
abs(100.12) :  100.12
abs(119L) :  119
```
