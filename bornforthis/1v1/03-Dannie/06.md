## 1. end=“\n”

### 1.1 代码1

```python
a = "Jaden"
b = "Austin"
c = "Dannie"
print(a)
print(b)
print(c)
```

输出：

```python
Jaden
Austin
Dannie
```

### 1.2 代码2

```python
a = "Jaden"
b = "Austin"
c = "Dannie"
print(a, end="\n")  # new line
print(b, end="\n")
print(c, end="\n")
```

输出：

```python
Jaden
Austin
Dannie
```

### 1.3 代码1 和代码2 是等价的！

代码1 是默认是换行。

> print 代码结束，默认是换行
