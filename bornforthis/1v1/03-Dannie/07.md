### 2.1 Dannie 编写的代码

```python
Austin='Coke'
Jaden='Juice'
Austin, Jaden= 'Juice', 'Coke'
print('Austin', Austin)
print('Jaden', Jaden)
```

**评价：这个地方你是直接拿 value 来进行赋值，算不上是交换果汁。**

自行修正的代码：

```python
Austin='Coke'
Jaden='Juice'
Austin, Jaden= Jaden, Austin
print('Austin', Austin)
print('Jaden', Jaden)
```
