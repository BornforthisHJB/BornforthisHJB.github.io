                        # 机身与镜头

                        不到换镜头的时候，就不要把镜头从相机上卸下来。

                        很多高端相机是不会配备闪光灯的，如果非要闪光灯就得另行购买。

                        # 相机的天敌

                        # 潮湿

                        # 防潮

                        晒晒太阳，晒掉霉菌等。

                        你受得了的阳光强度，相机就受得了。你受不了的，相机就别晒了。

                        # 水

                        相机落水，尽快取出电池！！！

                        不能烘干，烘干会导致相机变形，相机是精密仪器。放入大米中，也不要直接放，用透气的毛巾包裹放入。

                        从寒冷的地方到温暖的地方，就会产生。

                        一种是拔掉电池，慢慢回温。另一种放到包里面，等待回温。
